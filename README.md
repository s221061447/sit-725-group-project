### Server - NodeJS

The backend is built using NodeJS and will run on port 8000.

##### Installation

```bash
cd ./server
npm install
npm run start
# For dev environment
npm run dev
```

##### Dependencies

1. mongoose
2. express
3. jsonwebtoken
4. dotenv
5. bcryptjs
6. nodemon (dev-dependencies)
